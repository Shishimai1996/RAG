# Example

This document is a sample knowledge base for testing the RAG application.

## Overview
This app can answer questions based on the Markdown content stored in the data folder.

## Features
- It reads Markdown files from the data directory.
- It converts the content into vector embeddings.
- It retrieves the most relevant chunks when a user asks a question.

## Example Questions
You can ask questions like:
- What does this app do?
- Which files are used as knowledge sources?
- How does the RAG workflow work?

## Notes
The content here is intentionally simple so that you can test the embedding and retrieval flow easily.
If you add more detailed Markdown files, the answers will become more accurate and specific.
